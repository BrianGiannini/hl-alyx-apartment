===========

Whiteboard with three pens (red, green and blue), and an eraser. 
v1.0

How to use:
1. Ensure you have an updated hlvr.fgd
1. Drag .vmap file to <SDK>/content/hlvr/maps/prefabs
2. From within hammer, click prefab tab and drag whiteboard onto a wall and rotate if necessary.
3. Right click the prefab -> Selected prefabs -> Collapse
4. Add / Remote any markers or erasers as you desire.
